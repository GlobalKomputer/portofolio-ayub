my project

project 1 : membuat database rumah sakit

bulan kemarin saya membuat database rumah sakit yang bertujuan untuk mendukung kebutuhan informasi data pada rumah sakit

project 2 : membuat database pendaftaran

proses pembuatan database pendaftaran ini sangat penting untuk mengotomastiskan proses pendaftaran,verrifikasi dan pengiriman konfirmasi sehingga mengurangi kesalahan manual dan menghemat waktu

project 3 : membuat database perpustakaan

proses ini biasanya digunakan untuk memudahkan pencatatan dan pengelolaan buku, jurnal, dan materi perpustakaan lainnya termasuk informasi seperti judul, penulis, penerbit, no ISBN dan juga untuk mencatat buku yang dipinjam dan jatuh tanggal tempo, serta mengelola denda untuk keterlambatan pengemnbalian
